# API Reference

Base URL: `http://localhost:8000` (local) or your deployed backend origin. All routes except `/metrics` and `POST /api/auth/{register,login,refresh}` require `Authorization: Bearer <access_token>`. On a `401`, the frontend's axios interceptor automatically calls `POST /api/auth/refresh` and retries once.

Interactive OpenAPI/Swagger docs are auto-generated by FastAPI at `/docs` (Swagger UI) and `/redoc` whenever the backend is running — this file is a durable reference alongside that.

## Auth — `/api/auth`

| Method | Path | Auth | Request | Response |
|---|---|---|---|---|
| POST | `/register` | No | `{email, password, role, full_name, ...}` | `201` `TokenResponse` |
| POST | `/login` | No | `{email, password}` | `200` `TokenResponse` |
| POST | `/refresh` | No (refresh token in body) | `{refresh_token}` | `200` `TokenResponse` |
| POST | `/logout` | No (refresh token in body) | `{refresh_token}` | `200` `{"detail": "..."}` |

`TokenResponse`: `{access_token, refresh_token, token_type: "bearer", user: {...}}`. Errors: `401` invalid credentials, `409` email already registered.

## Error format

FastAPI's default `HTTPException` shape: `{"detail": "human-readable message"}`, with the corresponding HTTP status code (`400`, `401`, `403`, `404`, `409`, `422` for validation, `503` for an uninitialized subsystem — e.g. calling `/api/video/*` before `AgentService` finishes startup).
